Cloudflare Nameservers for All Domains
=========================================

All domains use the same nameservers:
- nora.ns.cloudflare.com
- wells.ns.cloudflare.com

Update these in GoDaddy for each domain:

1. cocoonspamini.com
2. glassbubble.net
3. iheartdinos.com
4. jughunt.com
5. lipbalmjunkie.com
6. ohsorad.com
7. rosamimosa.com
8. taicho.com
9. tokyo3.com

Instructions:
1. Log into GoDaddy
2. Go to DNS Management for each domain
3. Click "Change" next to Nameservers
4. Select "Custom"
5. Enter the two nameservers above
6. Save

After updating nameservers in GoDaddy, DNS will be fully managed by Cloudflare via Terraform.
